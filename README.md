# Messenger (Sprint 1)

Учебный проект в рамках курса «Мидл фронтенд-разработчик» от Яндекс Практикума.  
Проект представляет собой интерфейс мессенджера, реализованный с использованием **vanilla JavaScript**, **Handlebars** и **Vite**.

В первом спринте выполнена настройка сборки, создана структура проекта, шаблоны страниц и навигация между ними с помощью роутинга.

## Команды

Установка зависимостей:
```bash
npm install
```


Запуск в режиме разработки
```bash
npm run dev
```


Сборка проекта
```bash
npm run build
```
